# uma-project-tg-bot
Это бот для телеграмма, предоставляющий пользователю разного рода функционал: он может перевести текст с английского на русский, если вдруг случайно не переключил раскладку вовнемя, или наоборот - с русской раскладки на английскую, а так же сменить регистр с верхнего на нижний, если случайно написал большой текст с зажатым Caps Lock.
